How to use jobagent

jobagent is a job-application prep assistant. It does the busywork — finding roles, scoring them against your profile, drafting your materials and pre-filling what it safely can — then hands every application to you to review and submit.

🔒 What it will never do: submit an application to a real site · invent facts about you · answer screening questions on your behalf · ignore a site's robots.txt. The final, irreversible step is always yours.

Start in 30 seconds

Just exploring?

On the Dashboard, click Seed demo data. It loads a sample profile, résumés and a handful of roles, then runs the whole pipeline so finished applications appear in the review queue right away — completely offline, nothing sent anywhere.

What each tab is for

DashboardAt-a-glance counts, run the pipeline, and seed demo data.
JobsEvery role found, ranked by score. Prepare one to override the scorer, or Skip it.
Review queueThe human gate — finish each prepared application and record that you submitted.
TrackerApplications you've submitted and the outcome you record for each.
MaterialsYour résumés and cover letters. The agent only selects these — you author them.
CompaniesThe watchlist the pipeline sources roles from; activate or deactivate any entry.
ProfileThe facts the agent is allowed to use about you. Better profile → better prep.

The review queue is the human gate

This is the heart of the design. The agent prepares everything it safely can, then stops. On each application you'll see the chosen résumé, the rendered cover letter, the fields it pre-filled, and — separately — the fields only you can answer (screening questions, anything not in your profile).

You complete those, submit on the company's own site, and click Record submission. That button only logs that you submitted — there is no code path anywhere that sends an application for you.

Tips & troubleshooting

Ran the pipeline but nothing got queued?
The banner tells you why — usually a missing résumé or an incomplete profile. Add a résumé in Materials or complete your Profile, then run again.
Want a role the scorer skipped?
Open Jobs and click Prepare on it — that overrides the score and runs tailor + prep just for that role.
Scores feel basic?
Without a key, scoring uses a fast offline heuristic. Set ANTHROPIC_API_KEY (and install the anthropic SDK) to enable Claude-quality scoring and rationales.
Something looks broken?
Visit /healthz for diagnostics — it reports the database driver, connectivity, and whether templates loaded.
